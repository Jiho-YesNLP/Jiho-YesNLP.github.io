### Kennesaw State University

**CS 4267 / 7267 — Machine Learning**
Introduction to machine learning algorithms and applications, covering supervised and unsupervised learning, neural networks, deep learning, and reinforcement learning.
*Offered: Sp22, Sp23, Fall23, Sp25*

**CS 4635 / 7635 — Information Retrieval**
Techniques and algorithms for searching and retrieving information from large data collections: indexing, query processing, relevance feedback, text mining, and search engine design.
*Offered: Fall21, Fall22, Fall23, Fall24*

**CS 3410 — Database Systems**
Design, implementation, and management of databases: relational models, SQL, data modeling, and database security.
*Offered: Sp22, Sm22*
